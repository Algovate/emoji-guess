# 猜猜团 Emoji Guess — 隐私政策

最后更新：2026-07-04

猜猜团（Emoji Guess，以下简称“本扩展”）尊重并保护你的隐私。

### 1. 我们不收集任何数据
本扩展完全离线运行，不连接任何服务器，不内置分析、广告或第三方 SDK。我们不收集你的姓名、邮箱、位置、浏览历史，或任何其他个人身份信息。

### 2. 你的数据只存在你自己的设备上
你的游戏进度、等级、经验、成绩、连续挑战天数、每日挑战记录与本地排行榜，全部通过浏览器的本地存储（`chrome.storage`）保存在你**自己的设备**里，不会上传，也不会被我们或其他任何人读取。

### 3. 使用的权限及用途
- `sidePanel`：在浏览器侧边栏中显示游戏界面。
- `storage`：在本地保存你的进度与成绩。
- `clipboardWrite`：仅当你主动点击“复制”时，把题目或成绩写入剪贴板，以便你自行分享。剪贴板内容不会被读取或上传。

### 4. 权限范围之外
本扩展不会读取、修改或传输你访问的任何网页内容，也不申请任何网站访问权限（`host_permissions`）。

### 5. 卸载即彻底删除
卸载本扩展后，本地保存的所有数据将被浏览器一并清除。

### 6. 联系方式
如对本政策有疑问，可在 Chrome Web Store 的扩展页面留言或通过商店提供的联系方式反馈。
