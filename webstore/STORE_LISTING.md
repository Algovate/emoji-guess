## 详细描述

```
猜猜团是一款藏在 Chrome 侧边栏里的 Emoji 猜词小游戏，点一下图标就能开玩，一分钟一局，随时随地收获一点小开心。

【怎么玩】
看一组 Emoji，猜出它代表的词、成语、电影、歌曲或网络热词。
- 简单题：四个选项里点一个。
- 普通及以上：点击字块，拼出正确答案。

【四种模式】
- 限时：60 秒内尽量多答对，速度越快分越高。
- 闯关：每关 10 题，答对 8 题过关，难度递增。
- 无尽：三条生命，一直猜下去，挑战最高分。
- 每日挑战：每天同一组 10 题，比谁更快。

【题库】
500 道独立策划的题目，覆盖名词、动词、成语、电影、歌曲、明星、城市、节日、网络热词等。每道题答完都有脑洞解释，猜错也能会心一笑。

【更好玩的小细节】
- 三级提示：字数 → 首字 → 逐字揭晓。
- 速度奖励、连击倍率，越顺越上头。
- 本地等级、经验、连续挑战天数、历史最佳与每日正式成绩。
- 三套贴纸主题，可切换。
- 支持减少动态效果与键盘操作。
- 一键把题目或成绩复制成微信文字，或保存为图片分享卡片。

【你的数据只留在你这里】
完全离线，不需要注册，不联网，不上传任何数据。你的进度、成绩和排行榜只保存在你自己的浏览器里。

打开方式：点击工具栏里的“猜猜团”图标，侧边栏即刻打开。
```

## 权限说明
> 提交时 Dashboard 会问“为什么需要这些权限”。填：

| 权限 | 用途 |
|---|---|
| `sidePanel` | 在浏览器侧边栏中显示游戏界面。 |
| `storage` | 在本地保存玩家的等级、经验、成绩、连续天数与每日挑战进度。 |
| `clipboardWrite` | 仅当点击“复制”时，把题目或成绩写入剪贴板，方便自行分享。剪贴板内容不会被读取或上传。 |

未申请 `host_permissions`，不读取/修改任何网页内容。

### 重新生成截图
```bash
npm run build
# 若本机已装 playwright：
node webstore/capture-screens.mjs
# 否则用缓存里的 playwright（不修改 package.json）：
PLAYWRIGHT=/Users/rodin/.npm/_npx/e41f203b7505f1fb/node_modules/playwright node webstore/capture-screens.mjs
```
脚本会输出 `webstore/screens/*@2x.png`，再用 ImageMagick 缩到 1280×800 并去掉 alpha（脚本如需可扩展自动缩放）。
